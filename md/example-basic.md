## ย่อหน้าธรรมดา

นี่คือตัวอย่างเอกสาร Markdown ที่แปลงเป็น PDF ด้วย **md2pdf-th** รองรับภาษาไทยและอังกฤษ

Features หลัก:
- ภาษาไทย/English mixed content
- เลขหน้าอัตโนมัติ
- Header แสดงชื่อเรื่อง
- ตารางจัดเรียงไม่ล้นหน้า

## รายการ (Lists)

### ลำดับ

1. ติดตั้ง Node.js
2. รัน `npm install -g md2pdf-th`
3. แปลงไฟล์ `md2pdf document.md`

### ไม่ลำดับ

- React
- Vue
- Svelte
- Angular
